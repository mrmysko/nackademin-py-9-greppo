```bash

hopphej hej hopp
$ echo $? # → 0
```

### Tips

- Kom ihåg, arean av en rektangel beräknas som `längd * bredd`.

- Se till att din funktion skriver ut det krävda meddelandet innan den
  returnerar arean.
